# Estruturas de Repetição Avançadas com While em Python

Repositório com exercícios desenvolvidos para praticar estruturas de repetição utilizando o laço `while` em Python.

## Exercícios

### 1. Cálculo de Fatorial
Programa que calcula o fatorial de um número inteiro natural.

### 2. Quadrados Perfeitos
Programa que exibe os quadrados perfeitos menores que 20.

### 3. Desenho de Quadrado
Programa que desenha um quadrado utilizando caracteres `*`.

### 4. Escada de Asteriscos
Programa que exibe uma sequência crescente de asteriscos.

### 5. Soma de Série com Fatorial
Programa que calcula uma série matemática utilizando fatoriais.

## Assuntos praticados

- Laço `while`
- Loops aninhados
- Estruturas de repetição
- Fatorial
- Operações matemáticas
- Acumuladores
- Contadores
- Manipulação de padrões
- Lógica de programação
